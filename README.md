# SafeClickApp

SafeClickApp is a mobile application prototype focused on real-time notifications, location tracking, and interactive maps. The project highlights front-end interactivity and dynamic user interface features implemented with JavaScript.

## Features Implemented

- **User accounts** – creation and management of user profiles.  
- **Email verification** – ensures valid user accounts.  
- **Real-time location tracking** – displays user positions dynamically on a map.  
- **Interactive maps** – allows users to view locations with map navigation.  
- **Push notifications** – instant notifications between users.  
- **Front-end interactivity** – implemented with JavaScript for responsive UI.  
- **Ongoing UX improvements** – interface and user experience refinements in progress.  

## Technology Stack

- **Front-end:** HTML, CSS, JavaScript  
- **Mapping & Geolocation:** Interactive map libraries (e.g., Google Maps API or Leaflet)  
- **Notifications:** Push notification system  
- **Version Control:** Git/GitHub  

## Project Status

- Prototype under development  
- Core features implemented  
- UX and interface enhancements ongoing  

## Installation & Running

1. **Clone the repository**:

git clone git@github.com:matildeoliveirasilvafeup/Mobile-App-Development.git

2. **Navigate to the project folder**:

cd SafeClickApp


3. **Install dependencies**:

npm install


4. **Start the development server**:

npm start


5. **Open the app on a mobile device using Expo Go**

Scan the QR code displayed in the terminal or browser

The app will load on your device for testing
